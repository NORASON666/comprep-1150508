✕ ERROR EXAMPLE

2-1-1 核心素養 — 四種常見踩雷

CHECK 2-1-1 ／ 對應 P14–P15
條文：「核心素養」欄需填寫總綱層級指標（如「E-B2 科技資訊與媒體素養」）。國小校訂資訊課不能填寫科技領域綱要的指標（如「科-J-A1」），因為國小階段沒有資訊／科技領域。
狀況核心素養欄填寫內容
踩雷 ①只寫項目層
B2
✕ 缺階段層碼（E-）+ 完整敘述。「B2」是含國小／國中／高中三個階段的項目層、看不出針對國小階段。
踩雷 ②用不存在的代碼
科-E-A1
科-E-B2 科技資訊與媒體素養
✕「科-E」代碼根本不存在。國小無科技領域。常因誤用《科技領域草案》造此代碼。國中正確寫法是「科-J-A1」（J = 國中）、不是「科-E」。
踩雷 ③缺完整敘述
E-B2
✕ 階段層碼有了、但缺整段敘述。教育部要求總綱層級寫法是「代碼 + 敘述全文」（裕勝校長 4/24 共識）。
踩雷 ④用「資-E-」前綴
資-E-A2　具備探索問題的思考能力⋯
✕「資-E-」這個前綴根本不存在。國小階段沒有資訊／科技領域、不存在「資訊領綱核心素養」這種東西。
※ 若單元真的跨自然領域、可在總綱 E-XX 之外加上「自-E-A2 具備探索⋯」— 自然領綱核心素養是真實存在的、跨域時引用是合法寫法。
合格範例階段層 + 全文
E-B2 具備科技與資訊應用的基本素養，並理解各類媒體內容的意義與影響。
✓ 過 2-1-1 條文。階段碼 E-（國小）+ 完整敘述、來自 12 年國教課綱總綱階段層。
⚠ 一句話 RULE OF THUMB

國小校訂資訊課的核心素養 = 總綱階段層 E-XX + 完整敘述。看到「B2」「科-E」「資-E」就是錯。

依據：14 項檢核指標 2-1-1 ／ 4-24 港湖區共備裕勝校長提醒
